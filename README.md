# time-machine

A Node.js server for timezone conversion.



## Prerequisites

- Node.js v14+



## How to run

1. Install the dependencies:

```bash
npm install
```

2. Create `.env` file in the project root
3. Set up the following environment variables in `.env`:

```bash
NODE_ENV=local
PORT=4000
```

4. Execute [index.js](./index.js):

```bash
node index.js
```
